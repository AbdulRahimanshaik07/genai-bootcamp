Each day includes 4-6 hours of interactive sessions
Fundamentals
01

Foundation & Introduction

  • Introduction to Gen AI: History, evolution, and current landscape
  • Core Concepts: Build an AI-Powered App (Nano Banana Tool) Students will create a simple AI-powered app using the Nano Banana tool in Google AI Studio. This activity helps students understand how to design and deploy a basic AI model into an interactive app.
  • Tools Overview:Generate Images with Prompts,Develop an AI-Powered Chatbot This session introduces image generation using AI. Students will learn how to generate creative and realistic images by providing short text prompts.
  • Hands-on Session: First interactions with AI assistants,Analyze Images with AI Students will use Google AI Studio tools to analyze images — identifying objects, patterns, or attributes
Google AI Studio & Napkin AI
02

Google AI Studio & Napkin AI

  • Introduction to Napkin AI:Napkin AI and Visual Generation Students will get an overview of Napkin AI and learn how to create visuals using simple text prompts. They will explore how AI can turn written ideas into creative images.
  • Core Concepts:Transcribe Audio to Text Students will explore AI-based transcription tools that convert spoken language into text. This introduces how AI can process audio inputs for documentation or analysis purposes.
  • Tools Overview:Generate Creative Visuals Using Prompts,Design Creative Posters and Concepts with AI
  • Optimization: Iterative refinement and best practices
Notebook LLM
03

Notebook LLM

  • Introduction to Notebook:Getting Started with Notebook LLM Students will learn the basics of using Notebook LLM — understanding the platform interface, uploading files, and exploring its key features like chat, summaries, and audio tools.
  • Core Concepts:Upload and Analyze Study Material Students will learn how Notebook LLM processes study materials to identify important concepts and generate insights, helping them grasp how AI assists in content understanding.
  • Content Creation: Writing, documentation, and creative tasks,Generate Audio or Podcasts from Notes Students will explore how Notebook LLM can convert written text into audio, learning how AI supports auditory learning and enhances accessibility.
  • Case Studies: Real-world implementations and success stories,Visualize Concepts with Mindmaps,Ask and Learn — AI-Powered Q&A
Botpress, Gamma, and HuggingFace
04

Botpress, Gamma, and HuggingFace

  • Introduction to Botpress:Students will get an overview of Botpress and understand its use in creating AI-driven conversational chatbots for various applications.
  • Core Concepts:Build Your First Chatbot with Botpress,Create AI-Powered Presentations with Gamma
  • Development Tools: LangChain, OpenAI SDK, and workflow builders
  • Hands-on Build: Start your first AI application
Teachable Machine
05

Teachable Machine, RoboFlow

  • Introduction to Teachable Machine, RoboFlow, and MediaPipe Studio: Students will learn how to create and train a basic AI model using Teachable Machine to classify images or gestures through a simple, interactive interface
  • Core Concepts: Build a Basic Image Detection Project using RoboFlow,Students will understand how to use RoboFlow to implement an image detection or object recognition project with pre-trained models.
  • Development Tools: Performance tuning and cost management.AI vision-based mini project, showcasing their understanding of all three tools.
  • Project Continuation: Refine and enhance your applications
Advanced Topics
06

Advanced Projects & Development

  • Advanced Features: Multi-agent systems and complex workflows
  • Ethics & Safety:Responsible AI, bias mitigation, and privacy
  • Optimization: Performance tuning and cost management
  • Project Continuation: Refine and enhance your applications

What is Generative AI?

Generative AI uses machine learning models to produce new content—text, images, code, audio—based on patterns learned from data. It accelerates ideation, prototyping, and automation while augmenting human creativity.
